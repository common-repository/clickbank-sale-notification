ClickBank Sale Notification plugin will automatically send you an email notification every time there is a transaction in your ClickBank account.

== Description ==

ClickBank Sale Notification plugin will automatically send you an email notification every time there is a transaction in your clickBank account. Its support any type of transactions ( Sale, Rebill, Refund, Chargeback, Cancel Rebill ) and it also works for both affiliate and product vendor.  

How to Get ClickBank Secret Key?

1. Go to your ClickBank Account Settings &raquo; My Site &raquo; Advanced Tools Editor.
1. Click the Edit link in the top right corner of Advanced Tools Editor.
1. Create your Secret Key – up to 16 letters/digits ALL CAPS.
1. Fill in the Instant Notification URL with: http://YOURSITE.com/notify.php.
1. Use the Secret Key you created before in the plugin settings.

== Installation ==

1. Download it here. or search from WordPress Admin > Plugin > Add New.
1. Install and activate from WordPress Admin > Plugins page.
1. Go to the WordPress Admin Options page > ClickBank Sale and fill in your ClickBank Secret Key and email address.
